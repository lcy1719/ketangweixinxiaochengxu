### 万字文档+在线课堂小程序+Uniapp+SSM+Vue+sql文件+软件环境安装包
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、Tomcat7.5-9、微信开发者工具/HBuilderX
技术栈：Java、MySQL、Tomcat7.5-9、Spring、SpringMVC、Mybatis/MybatisPlus、Vue2、Uniapp、ElementUI、HTML5（H5）、JavaScript、CSS、图片上传
采用M（model）V（view）C（controller）三层体系结构，MySQL数据库作为系统数据储存平台，实现了基于B/S结构的Web系统。

系统功能模块：
本系统包含管理员和用户2个角色。
管理员角色：
1. 用户管理：管理系统中所有用户的信息，包括添加、删除和修改用户。
2. 课程分类管理：管理课程分类信息，包括添加、删除和修改课程分类。
3. 课程信息管理：管理课程信息，包括添加、删除和修改课程信息。
4. 课程订阅管理：管理课程订阅信息，包括查看订阅详情、推送视频、修改和删除订阅。
5. 课程视频管理：管理课程视频信息，包括添加、删除和修改课程视频。
6. 公告栏管理：管理公告栏信息，包括添加、删除和修改公告，以及查看评论。
7. 留言板管理：管理留言板信息，包括查看留言详情、修改、回复和删除留言。
8. 系统管理：管理系统配置，包括轮播图管理。
用户角色：
1. 注册与登录：用户可以通过注册账号并登录系统。
2. 课程信息查看：用户可以查看课程信息，并进行课程订阅和收藏操作。
3. 公告栏查看：用户可以查看公告栏信息，并提交评论。
4. 个人信息管理：用户可以管理自己的个人信息，包括修改个人信息和退出登录。
5. 课程订阅：用户可以订阅课程，并查看订阅详情。
6. 我的页面：用户可以查看课程信息、课程订阅、课程视频、我的收藏管理、在线客服、留言板等信息。
